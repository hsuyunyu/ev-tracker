養車小幫手 (Car Maintenance Helper) 🚗⚡

一款輕量級、無需後端、完全離線可用的網頁應用程式 (PWA)。專為電動車及燃油車主設計，協助您輕鬆紀錄並分析所有車輛相關花費。

✨ 主要功能

1. 全方位記帳

* 多種花費類別：支援 充電、停車、過路費、保養、保險 及 其他。

* 智慧欄位：根據選擇的類別自動切換欄位（例如：選「充電」顯示度數；選「保養」顯示里程）。

* 分帳功能：支援紀錄不同的駕駛/使用者，方便家庭共用車輛時統計花費。

* 自動記憶：輸入過的廠商或地點會自動記憶，下次輸入時提供建議。

2. 視覺化儀表盤

* 統計全覽：即時計算總花費、總充電度數、平均每度電費。

* 靈活篩選：支援 依年份、依月份 或 自訂日期範圍 進行分析。

* 圖表分析：

    * 花費類別佔比 (圓餅圖)

    * 使用者分帳統計 (圓餅圖)

    * 花費趨勢 (長條圖)：依據篩選範圍自動調整顯示密度 (日/月)。

3. 車輛管理

* 多車支援：可新增多台車輛資料 (廠牌、型號、電池容量、發照日等)。

* 里程追蹤：紀錄最近一次的里程數，自動計算車齡。

* 保險提醒：在車輛卡片顯示保險到期日，少於 30 天會以紅字警示。

4. 資料安全與備份

* 隱私優先：所有資料皆儲存於您瀏覽器的 Local Storage 中，不會上傳至任何伺服器。

* 備份還原：支援匯出完整資料為 JSON 檔，方便備份或轉移至其他裝置。

*  Excel 匯出：可將紀錄匯出為 CSV 格式，方便在 Excel 中進行進階編輯。

5. 個人化體驗

* PWA 支援：在手機上可透過「加入主畫面」安裝為 App，支援全螢幕操作。

* 深色模式：內建亮/暗主題切換，夜間紀錄不刺眼。

* 自訂顯示：可設定首頁顯示的最近紀錄筆數。

🚀 如何使用

*由於本程式為單一 HTML 檔案架構，使用非常簡單：

* 方法一：直接開啟

* 下載 ev_charging_tracker_react.html 檔案。

* 使用任何現代瀏覽器 (Chrome, Safari, Edge) 直接開啟該檔案即可使用。

方法二：手機 App 體驗 (推薦)

* 將檔案上傳至免費的靜態網頁託管服務 (如 GitHub Pages 或 Netlify Drop)。

* 使用 iPhone (Safari) 或 Android (Chrome) 開啟網址。

* 點擊分享按鈕，選擇 「加入主畫面」。

* 現在您可以像原生 App 一樣在全螢幕模式下使用它。

🛠️ 技術架構

* 核心框架: React 18 (透過 CDN 載入)

* 樣式庫: Tailwind CSS (透過 CDN 載入)

* 圖示庫: Lucide Icons

* 編譯器: Babel (瀏覽器端即時編譯 JSX)

資料儲存: Browser LocalStorage

📂 資料備份與轉移

* 因為資料儲存在瀏覽器快取中，若清除快取資料將會消失。建議定期備份：

* 進入 「設定」 頁面。

* 點擊 「備份資料 (下載)」 下載 .json 檔案。

* 若要轉移到新手機，在新手機開啟網頁，進入設定頁面點擊 「還原備份」 並選擇該檔案即可。
